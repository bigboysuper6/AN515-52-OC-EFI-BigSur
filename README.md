# AN515-52-OC-EFI-BigSur

机型：宏碁AN515-52 暗影骑士3
+ cpu:i5-8300h
+ gpu:1060
+ 网卡 bcm94352z 原来的intel网卡9560可以直接换
+ 声卡 alc255 

不工作：
+ 显卡1060
+ hdmi接口（外接显示器走displaylink方案，使用外置显卡，我在闲鱼买的是睿音usb3.0扩展坞）
+ 换网卡可以隔空投送不换就不能
